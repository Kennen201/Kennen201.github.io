install-domain controller en PREMIER PAR PITIE

la classique ip fixe, update manuel,découverte réseau, changer le nom + install adds

Etape 1 : préparation des près requis. 

Etape 1.1 : Ouvrir la console PowerShell en administrateur et taper la commande : 

Install-WindowsFeature RSAT-ADDS

Etape 1.2 : installer netframework 4.8 et une fois fait redémarrer. 

Etape 1.3 : monter l’iso sur le lecteur disque.

Etape 1.4 : ouvrir un CMD en administrateur et ce rentre dans le lecteur D


 

 

Etape 1.5 : taper les commandes suivantes : 

Setup.exe /PrepareSchema /IAcceptExchangeServerLicenseTerms_DiagnosticDataOFF

Setup.exe /PrepareAD /OrganizationName:TSSR /IAcceptExchangeServerLicenseTerms_DiagnosticDataOFF

Setup.exe /PrepareAllDomains /IAcceptExchangeServerLicenseTerms_DiagnosticDataOFF
