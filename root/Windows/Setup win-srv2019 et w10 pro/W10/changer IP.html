changer IP

Pour commencer, il faut ouvrir la fenêtre exécuter avec le raccourci “Win+R”

Dans cette fenêtre, rentrer la commande “ncpa.cpl”

Dans la fenêtre qui s'ouvre, cliquer sur la carte réseau souhaité

Ensuite sur “Propriétés”

Ensuite “Protocole Internet Version 4(TCP/IPv4)”

Dans la fenêtre qui s'ouvre, sélectionner “Utiliser l'adresse IP suivante” et rentrer l'adresse IP souhaité

On peut vérifier ensuite que l'IP à bien été changée en ouvrant une invite de commande en en tapant “ipconfig”
