1. changer nom

Pour commencer, il faut ouvrir la fenêtre avec le raccourci “Win+R”

Dans cette fenêtre, entrer la commande “sconfig”

A l'intérieur de cette interface, entrer “2” pour rentrer dans le changement de nom

Ici, rentrer le nom souhaité et un redémarrage est nécessaire pour appliquer les changements.
