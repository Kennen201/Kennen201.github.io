2. changer MàJ en manuelles

Pour commencer, il faut ouvrir la fenêtre exécuter avec le raccourci “Win+R”

Dans cette fenêtre, entrer la commande “sconfig”

Ici, rentrer “5” pour arriver aux paramètres de mises à jour

Ensuite, entrer “m” pour changer les mises à jour en manuelles, une fenêtre s'affichera pour confirmer les changements
