Créer un groupe GG (groupe global) 

Ensuite créer un groupe Domaine Local en Lecture seule ou Lecture/Ecriture

Ensuite, les utilisateurs devront être membre du Groupe Global (GG) et les GG devront être membre des Domaines Locaux, c'est avec eux qu'on fera les permissions.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Ensuite sur un ficher partagé : dans l'onglet partage, on choisit le DL pour que les permissions soit bien données et réglées, car en effet, les GG sont membres des DL qui vont nous être utile pour les permissions,  et lorsqu'on choisit une DL pour les permissions, elles sont automatiquement appliquées au dossier car les membres en font parti
