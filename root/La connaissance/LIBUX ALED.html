que trouve t-on dans :

boot : tout les fichiers de démarrage, le GRUB (grand unified boot order)

root : tout les fichiers correspondants au superutilisateurs ROOT

etc : tout les fichiers de configuration en format TXT 

home : tout les fichiers personnels de l'utilisateur (documents/dl/bureau etc)

 

/Etc/networ/interfaces/ configuration adressage ip

/etc/apt/sources/list: sert à modifier la source de téléchargements (apt) (déîts

/etc/sudoers : sert àg érer les privilèges des superutilistaeurs

 

créer un dossier : mkdir

créer fichier vide : touch

éditer fichier :  ano

contenu d'un dossier : ls  (et -l pour une LIST) (-a pour montrer tout -ALL)

connaitre ip : ip -c a

 

démarrer un service : systemctl start

déplacer à la racine : cd

déplacer dossier parent : cd ..

télécharge un élément : wget

 

stopper ma vm : shutdown -h (ou 0)

passer en root : su -

lancer un service au boot : systemctl enable

je veux les droits admin pour une commande : sudo 

 

commandes apt :

apt update : rechercher les mises à jours de tous les paquest

apt upgrade : mises à jour des paquets ressorti par la commande précédente

apt install : installer d'un nouveau paquet

apt purge : suppression d'un paquet installé
