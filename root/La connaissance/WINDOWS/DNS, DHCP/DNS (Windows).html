pq on l'utilise en entreprise

c'est plus simple de se souvenir d'un nom de domaine qu'une adresse IP, 

comment il bosse le frérot

il associe des noms de domaine avec leurs adresses IP
(exemple 142.250.179.78 dans l'URL d'un browser ramène sur google.com)

Le serveur DNS est composé d'enregistrements (records), il y en a plusieurs type, le point commun entre eux et qu'ils ont tous un TTL (Time To Live), ces principaux types sont :

Enregistrement A - L'enregistrement qui contient l'adresse IPv4 d'un domaine.

Enregistrement AAAA : même chose mais en IPv6

Enregistrement CNAME : (canonic name/nom canonique) utilisé lorsqu'un domaine est un alias d'un autre domaine ou sous domaine, elle ne renvoie pas vers des IPs.

Enregistrement MX : (Mail exchange/échange de mails) il dirige le courrier vers un serveur de courrier, comme le CNAME il pointe vers des domaines, pas des IPs. Il indique comment le courrier doit être envoyé conformément au protocole SMTP(Simple Mail Transfer Protocol)

La priorité dans ses mails fait que le courrier choisit le serveur avec la valeur de priorité la moins élevé, si le premier serveur ne répond pas, le courrier va sur le second. Ils peuvent aussi être de même valeur pour équilibrer la charge entre les serveurs.
