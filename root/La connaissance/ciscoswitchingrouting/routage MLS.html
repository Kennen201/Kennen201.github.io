routage MLS

sur un MLS (switch de niveau 3) connecté à plusieurs switchs, il faut que les VLAN soient crées dans le switch (pas besoin de mettre de range),  et que les interfaces qui vont vers des switch soient en trunk, et que les interfaces des switchs qui vont vers le MLS soient en trunk aussi

pour le MLS 

en
conf t
ip routing
vlan 10
name VLAN10
exit
vlan 20
name VLAN20
exit
vlan 30
name VLAN30
exit
interface g0/0
switchport nonegotiate
switchport mode trunk
ip address 192.168.10.254 255.255.255.0
exit
interface g0/1
switchport nonegotiate
switchport mode trunk
ip address 192.168.20.254 255.255.255.0
exit
interface g0/2
switchport nonegotiate
switchport mode trunk
ip address 192.168.30.254 255.255.255.0
exit
interface g0/3
switchport nonegotiate
switchport mode trunk
exit

pour le switch :

en
conf t
interface g0/1
switchport nonegotiate
switchport mode trunk
