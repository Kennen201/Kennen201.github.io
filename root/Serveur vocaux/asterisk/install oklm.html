install oklm

  1. Se connecter en SSH puis ce mettre en root
  2. Commandes : 

apt update

apt upgrade

  1. Se placer dans le dossier

cd /usr/src/

  1. Télécharger la version 16 ( il est possible que d’autre version soit disponible ici nous travaillons avec la v16)

wget  https://downloads.asterisk.org/pub/telephony/asterisk/asterisk-16-current.tar.gz

  1. On extrait l’archive

tar xzvf asterisk-16-current.tar.gz

  1. On va commencer l’installation, placer vous dans le dossier asterisk

cd asterisk-16.10.0

  1. Puis taper la commande 

contrib/scripts/install_prereq install

  1. Utilisation du préfixe 33

  1. Une fois les pré requis installé vous devez recevoir le message suivant : 

  1. On vérifie et on met à jour les pré requis pour les formats mp3 

contrib/scripts/get_mp3_source.sh

  1. Une fois fait 

./configure --with-jansson-bundled

  1. On s’attaque au plus gros parti 

make menuselect

  1. Une fois le menu suivant qui s’affiche bien faire attention 

Il faut choisir dans Add-ons :

[*] format_mp3

# pour les packages de sons, décocher tous les autres et ne prendre que les modules en français

Il faut choisir dans « Core Sound Packages » :

[*] CORE-SOUNDS-FR-ULAW

[*] CORE-SOUNDS-FR-ALAW

Il faut choisir dans « Music On Hold File Packages » :

[*] MOH-OPSOUND-ULAW

[*] MOH-OPSOUND_ALAW

Il faut choisir dans « Extras Sound Packages » :

[*} EXTRA-SOUNDS-FR-ULAW

[*] EXTRA-SOUNDS-FR_ALAW

faire "esc" puis "Save & Exit" et valider.


 

 

  1. Nous reste qu’a compilé

make -j2

  1. On lance l’installation 

make install

  1. Taper les commandes suivantes 

 

 

  1. La commande pour ouvrir la console Asterisk

asterisk –rvv

  1. Pour quitter la console 

Exit
