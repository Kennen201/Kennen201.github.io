Proc ajout contexte

  • Pour rajouter un contexte on va aller dans /etc/asterisk/extensions.conf
    • Dupliquer le fichier extensions.conf dans extensions.conf.original
    • > /etc/asterisk/extensions.conf
    • nano /etc/asterisk/extensions.conf
  • Un contexte est constitué de:
    • Son nom entre crochets
    • les actions contenue dans le contexte (les appels, la messagerie vocale, les conférences, etc…)
    • les contextes inclus via la commande include
  • On a donc juste besoin de rajouter le nom du contexte et une action pour qu'il soit créé.
  • un exemple de contexte créé:
  • [it]: le nom du contexte (forcément entre crochets)
  • include: une commande qu'on utilise pour inclure d'autres contextes pour pouvoir communiquer avec eux (ici on fait en sorte que it puisse communiquer avec compta)
  • le bloc de commandes encadrée en rouge: le déroulement de l'appel selon le serveur astérisk. (ouverture de ligne --→ sonnerie jusqu'à ce que l'interlocuteur décroche --→ raccroche auto si non réponse.)

 

  • N'oubliez pas de sauvegarder et de reloader Asterisk.
