Proc Configuration utilisateur

  • la configuration des utilisateurs (et donc des numéros) se situe dans /etc/asterisk/sip.conf
  • cependant, il vaut mieux copier ce fichier et le cloner de façon a garder l'original si c'est la première modif
  • maintenant, faire nano /etc/asterisk/sip.conf 
    • pour créer un utilisateur, il suffit juste de rentrer ceci dans sip.conf et de remplacer avec les informations de l'utilisateur que vous voulez créer:
  • Et voila !!! Maintenant sauvegardez et reloadez Asterisk.
    • Afin de pouvoir tester si les utilisateurs sont bien configurés, faites 2 utilisateurs.
    • voici le rajout en texte (pour pouvoir le copier coller a partir d'un cmd/terminal en ssh vers la vm)
    • Ne pas oublier de remplacer les champs concernés:

[XXX]                                           ;numéro  
secret=XXXX                                     ;mot de passe  
callerid= ‘NOM DE L’UTILISATEUR' <XXX>                        ;nom affiché  
mailbox=XXX@"nom du contexte"                             ;boite mail  
type=friend  
host=dynamic  
nat=no  
context=local                                   ;contexte dans lequel est l'utilisateur
