Proc config appel simple

  • Dans un premier temps, il nous faudra deux utilisateurs configurés et chacun présent dans le même contexte
  • Ensuite nous allons aller dans extensions.conf via la commande suivante:
    • nano /etc/asterisk/extensions.conf
  • ensuite nous allons marquer juste en dessous du nom du contexte les lignes suivantes:
    • exten => _1XX,  1, Answer()
    • exten => _1XX, n, Dial(SIP/${EXTEN},10)
    • exten => _1XX, n, Hangup()
  • Vous obtiendrez un résultat similaire a ça après votre saisie:
  • Chaque ligne qu'on a rajouté correspond a une étape d'un appel simple:
    • La première ligne correspond a une ouverture de ligne:
      • _1XX: Numéro appelé, on met des X pour éviter d'avoir a mettre ces lignes pour chaque numéro concerné. 
      • 1: la priorité 
      • Answer(): la commande d'ouverture de ligne. 
    • La seconde ligne correspond a l'envoi de l'appel au numéro appelé:
      • Dial(): la commande d'appel 
        • SIP/: on cherche dans nos SIP le numéro 
        • ${EXTEN}: on reprend le numéro qu'on a appelé 
        • 10: la durée de la sonnerie en secondes 
    • La dernière ligne est ce qui raccroche si il n'y a aucune réponse.

 

  • Et voila !! votre configuration d'appel est prête, sauvegardez, reloadez asterisk et testez en appelant un numéro d'un utilisateur configuré !
