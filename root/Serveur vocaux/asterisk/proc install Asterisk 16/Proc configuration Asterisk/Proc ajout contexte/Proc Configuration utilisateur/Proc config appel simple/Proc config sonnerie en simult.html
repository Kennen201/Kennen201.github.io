Proc config sonnerie en simultanée

  • Pour faire sonner a en simultané des numéros, on va configurer de la même manière que pour un appel simple, mais en modifiant la ligne:
    • exten => _1XX, n, Dial(SIP/${EXTEN},10) -------→ exten => 1001, n, Dial(SIP/XXX&SIP/XXX&…,10)
      • 1001: numéro utilisé pour appelé en simultané
      • on remplace SIP/${EXTEN} par SIP/XXX plus un symbole & (=et)
