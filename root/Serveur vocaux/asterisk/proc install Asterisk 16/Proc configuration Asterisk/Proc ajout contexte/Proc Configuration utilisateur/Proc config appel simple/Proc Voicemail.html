Proc Voicemail

  • Pour mettre en place des messageries vocale pour nos utilisateurs, il va nous falloir modifier les fichiers /etc/asterisk/voicemail.conf et /etc/asterisk/extensions.conf .
  • on va commencer par /etc/asterisk/voicemail.conf:
    • on duplique d'abord voicemail.conf dans voicemail.conf.original ( cp /etc/asterisk/voicemail.conf /etc/asterisk/voicemail.conf.original )
    • on vide voicemail.conf (> /etc/asterisk/voicemail.conf )
    • on va dans le fichier via la commande nano /etc/asterisk/voicemail.conf
    • on va rajouter les lignes suivantes:
  • format: le format de fichier utilisé pour les messages vocaux
  • serveremail: l'adresse email pour le serveur
  • maxsilence: temps maximal de silence en secondes avant que la messagerie raccroche
  • silencethreshold: indique la sensibilité de micro pour détecter un silence potentiel, moins le chiffre est gros, plus c'est sensible.
  • maxlogins: le nombre de tentatives de connexion a la messagerie vocale avant que ça soit annulé.

 

  • Ensuite, on indique le contexte (ici local):
    • on indique ensuite la ligne suivante pour créer une boite vocale:
      • XXX => ,"NOM DE L'UTILISATEUR"
        • XXX: le numéro de l'utilisateur a qui appartiendra la boite vocale
  • On sauvegarde et on quitte.
  • dans sip.conf, s'assurer que le paramètre hasvoicemail est sur yes
  • Enfin, on va modifier la séquence d'appel pour que la personne qui appelle soit redirigé vers une messagerie vocale si personne ne répond en rajoutant la ligne suivante:
  • remplacer 1XX par ce qui a été mit comme numéro dans votre séquence d'appel (ici c'est 1XX)
  • s'assurer que la priorité (n) est correcte

  • sauvegardez et reloadez asterisk puis testez en appelant un numéro sans répondre puis en appelant le 109.
