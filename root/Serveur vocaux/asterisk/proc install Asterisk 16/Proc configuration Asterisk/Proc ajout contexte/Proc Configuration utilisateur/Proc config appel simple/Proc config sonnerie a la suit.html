Proc config sonnerie a la suite

  • Pour faire sonner a la suite des numéros, on va configurer de la même manière que pour un appel simple, mais en modifiant la ligne:
    • exten => _1XX, n, Dial(SIP/${EXTEN},10) -----→ exten => 1000, n, Dial(SIP/XXX,10)
      • 1000: correspond au numéro qu'on utilisera pour faire sonner a la suite un groupe (contexte) d'utilisateurs configuré a l'avance
      • on a remplacé ${EXTEN} par XXX qui sera un numéro dans notre groupe
    • Puis on répètera cette ligne pour chaque numéro de notre groupe qu'on veut faire sonner après celui la.
