Proc conférences

  • On va créer trois utilisateurs et deux salles pour lancer des conférences.
    • Pour ça on va dupliquer confbridge.conf dans confbridge.conf.original:
      • cp /etc/asterisk/confbridge.conf /etc/asterisk/confbridge.conf.original
    • On vide confbridge.conf:
      • > /etc/asterisk/confbridge.conf
    • On va dans confbridge.conf
      • nano /etc/asterisk/confbridge.conf
    • On rajoute les lignes suivantes:
  • On a donc créé un utilisateur non authentifié ( User_NoAuth ), un utilisateur authentifié ( User_Auth ), un utilisateur admin ( User_Admin ) et deux salles ( ConfRoom_1 et ConfRoom_2 ) pour les conférences.
  • on sauvegarde et on ferme le fichier.
  • Maintenant, on va dans extensions.conf:
    • nano /etc/asterisk/extensions.conf
  • on rentre les lignes suivantes:
  • 1002/1003: les numéros assignés la première salle (ConfRoom_1):
    • 1002 sera le numéro pour rejoindre la salle en tant qu'utilisateur authentifié
      • 1003 sera le numéro pour rejoindre la salle en tant qu'admin
  • 1004/1005: même chose mais pour la deuxieme salle (ConfRoom_2)
  • On sauvegarde, on reload Asterisk et on teste.
