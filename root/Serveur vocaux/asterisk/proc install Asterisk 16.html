proc install Asterisk 16

  • Lancer le support sur lequel on va installer Asterisk 16 (ici, ça sera une VM Debian 10 sans install graphique avec SSH)
  • installer Filezilla sur sa machine physique
  • trouver l'ip de notre VM:
    • ip -c a
    • vous obtiendrez le résultat suivant:
  • Ici, notre IP est le 192.168.240.131

 

  • Ouvrir Filezilla, se connecter a notre VM avec l'ip, le nom de l'utilisateur, son mot de passe et le port 22 (port SSH)
  • déplacer l'archive d'asterisk 16 dans notre VM par Filezilla (bien faire attention a le déplacer dans un dossier auquel l'utilisateur a accès, le mieux est /home/<NOM D'UTILISATEUR> )
  • déconnecter et fermer Filezilla
  • sur notre VM, mettre a jour la liste des paquets:
    • apt update
    • apt upgrade
  • se mettre dans le répertoire ou on a placé notre archive (ici je l'ai mis dans /home/user/):
    • cd /home/user/
  • extraire l'archive de asterisk 16:
    • tar xzvf asterisk-16-current.tar.gz
  • on va maintenant aller dans le répertoire issue de l'archive extraite:
    • cd asterisk 16.XX.X ( XX.X représenté par notre numéro de version complet)
  • marquer contrib/scripts/install_prereq install (ce qui va lancer le début de l'installation)
  • Indiquer 33 comme préfixe UIT-T
  • Vous recevrez ce message quand la commande aura fini:
  • on va ensuite mettre a jour les pré-requis pour les formats MP3:
    • d'abord: contrib/scripts/get_mp3_source.sh
    • ensuite: ./configure --with-jansson-bundled
  • Maintenant que les pré-requis sont a jour, on va régler des paramètres:
    • make menuselect
  • On va cocher ce qu'on veut activer/ajouter
    • add-ons --→ format_mp3
    • Core sounds packages --→ décocher CORE-SOUNDS-EN-ALAW et cocher CORE-SOUNDS-FR-ALAW / CORE-SOUNDS-FR-ULAW
    • Music On Hold File packages --→ décocher MOH-OPSOUND-WAV et cocher MOH-OPSOUND-ALAW / MOH-OPSOUND-ULAW
    • Extras sounds packages --→ EXTRA-SOUNDS-FR-ALAW / EXTRA-SOUNDS-FR-ULAW
  • ESC --→ Save & exit
  • Et maintenant, on compile Asterisk:
    • make -j2
  • puis on lance l'install:
    • make install
    • make samples
    • make config
    • ldconfig
    • make install-logrotate
  • enfin, je lance Asterisk:
    • systemctl start asterisk
    • systemctl status asterisk
  • je lance la console d'asterisk:
    • asterisk -rvv
    • pour sortir de la console: exit

 

  • Et voila ! Asterisk est installé.
