Mise en place du proxy

On va dans Services puis Squid Proxy Server

 

Local Cache

 

On doit tout d'abord commencer par régler le Local Cache.

 

On change la valeur de Hard Disk Cache Size pour mettre une valeur plus haute

 

On fait la même chose pour Memory Cache Size

 

On descend tout en bas et on clique sur Save

 

General

 

On peut maintenant effectuer la configuration génerale

 

On coche la case Enable Squid Proxy

 

On coche la case Resolve DNS IPv4 First

 

On coche la case Transparent HTTP Proxy

 

On coche la case SSL filtering, on change SSL/MITM Mode pour Splice All et on donne notre CA (Autorité de certification)

 

On coche Suppress Squid Version

 

On descend tout en bas et on clique sur Save
